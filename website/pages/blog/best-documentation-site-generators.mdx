To choose the best documentation site generators I picked the ones that had

-   Ease of use
-   Markdown support ([MDX](https://mdxjs.com) is a plus)
-   Pretty without effort
-   Integration with other tools
-   Dark mode

# 1. [Dokz](https://github.com/remorses/dokz)

![](/blog/best-generators/dokz.jpg)

Dokz is what powers the post you are reading, this probably means it has even good SEO

You can find dokz used in the wild in projects like [genql](https://genql.now.sh/docs) and [landing-blocks](https://landing-blocks.now.sh/docs)

The main features are

-   MDX support
-   Dark mode
-   Can be integrated with existing website
-   Deploy in 2 minutes with [Vercel](https://vercel.com/)
-   Themeable code snippets
-   React components Playground
-   Blog built in

# 2. [Docusaurus](https://docusaurus.io/)

![](/blog/best-generators/docusaurus.jpg)

Docusaurus powers many facebook open source projects like [react](https://it.reactjs.org/) and [react native](https://reactnative.dev/docs/getting-started)

-   MDX support
-   Dark mode
-   Document Search
-   Versioning support
-   Blog built in

# 3. [Mkdocs with material theme](https://github.com/squidfunk/mkdocs-material)

![](/blog/best-generators/mkdocs.jpg)

Mkdocs is a tool written in python that takes markdown files and generates a documentation website

Mkdocs default theme is not very good, but using the material theme the look is pretty good

You can find many python projects that use mkdocs, some examples are [starlette](https://www.starlette.io/) and [uvicorn](https://www.uvicorn.org/)

# 4. [VuePress](https://vuepress.vuejs.org/)

![](/blog/best-generators/vuepress.jpg)

VuePress is very good if you are already using vue components for you website, you can use them in your documentation too

-   Vue components in markdown
-   Generates a lightweight static website

# 5. [Doctocat](https://github.com/primer/doctocat)

![](/blog/best-generators/doctocat.jpg)

Doctocat is a gatsby theme developed by Github to power the [github documentation](https://help.github.com/en)

However you need to learn how to use Gatsby themes before using Doctocat proficiently

# 6. [Docup](https://github.com/egoist/docup)

![](/blog/best-generators/docup.jpg)

Docup is a single JavaScript file that fetches Markdown file and renders it as a beautiful one-page documentation.

Docup is built with Preact, the entire bundle (with CSS) is just 30kB minified and gzipped.
