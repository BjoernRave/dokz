# Introduction

Dokz enables you to quickly create live-reloading, seo-friendly, production-ready documentation sites with MDX.
It is based on `Next.js` so you get all the awesome features of Nextjs.

This entire site is built with `Dokz` starting from some markdown files ðŸ”¥

# Features

### React components in markdown (MDX)

Mdx let you import react components in your documentation markdown to enrich the user experience.

You can use any react component, for example the following is a chart component from `victory` charts

<Box fontSize='14px' maxW='500px'>
    <VictoryChart theme={VictoryTheme.material} domainPadding={20}>
        <VictoryAxis
            tickValues={[1, 2, 3, 4]}
            tickFormat={['Quarter 1', 'Quarter 2', 'Quarter 3', 'Quarter 4']}
        />
        <VictoryAxis dependentAxis tickFormat={(x) => `$${(x / 1000)}k`} />
        <VictoryBar
            data={[
                { quarter: 1, earnings: 13000 },
                { quarter: 2, earnings: 16500 },
                { quarter: 3, earnings: 14250 },
                { quarter: 4, earnings: 19000 },
            ]}
            x='quarter'
            y='earnings'
        />
    </VictoryChart>
</Box>

### Preview react code

Dokz let you preview react code indie code blocks with the `live=true` attribute

```js live=true
<div style={{ height: 100, background: 'lightblue' }}></div>
```

### Automatic sidebar and table of contents

The sidebar is generated based on your markdown files inside the `pages` directory
The table of contents on the right is generated based on the heading elements inside the markdown document
