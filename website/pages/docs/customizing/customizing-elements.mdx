# Customizing Elements

You can customize your Dokz site logo, nav items and footer, simply pass the elements to the `DokzProvider` props:

```jsx
import { DokzProvider, GithubLink, ColorModeSwitch } from 'dokz/dist'
import React from 'react'

export default (props) => {
    const { Component, pageProps } = props
    return (
        <DokzProvider
            headerItems={[
                <GithubLink url='https://github.com/remorses/dokz' />,
                <ColorModeSwitch />,
            ]}
            headerLogo={
                <img
                    src='/dokz_logo.svg'
                    style={{ opacity: 0.8 }}
                    width='100px'
                />
            }
            initialColorMode='dark'
            sidebarOrdering={{
                'index.mdx': 1,
                general: {
                    'getting_started.mdx': null,
                    'writing_mdx.mdx': null,
                },
                customizing: { 'customizing_components.mdx': null },
            }}
        >
            <Component {...pageProps} />
        </DokzProvider>
    )
}
```


Other customizations are possible via the `DokzProvider` props
